# Joyeux Anniversaire Alexis !

An interactive Star Wars-themed birthday card webapp built with love by the team.

## The Experience

1. **Click to Start** — Starfield with suspense
2. **Imperial Terminal** — DOS-style TUI with boot sequence, enter the secret code
3. **Blast Door** — Hydraulic door opens with SFX
4. **Trench Run** — 3-phase mini-game with real Star Wars sound effects
   - Phase 1: Enter the trench, dodge turbolaser fire
   - Phase 2: TIE Fighters arrive, wingmen get picked off
   - Phase 3: Use the Force — targeting computer, exhaust port, Han Solo saves the day
5. **"Il y a longtemps..."** — Classic blue text on starfield
6. **Logo Reveal** — "Joyeux Anniversaire Alexis" with Star Wars theme
7. **Crawl Text** — Personalized opening crawl
8. **Venator Construction** — 5,345 Lego bricks building the UCS Venator brick by brick
9. **Final Message** — Hologram screen with all contributors

## Tech Stack

- **SvelteKit** (static adapter)
- **Three.js** + GLTFLoader — Lego Venator 3D model
- **Canvas 2D** — Trench run mini-game
- **Howler.js** — Star Wars theme + sound effects
- **Web Audio API** — Synthesized terminal beeps, door sounds

## Cheat Codes

Type in the Imperial Terminal:
- `iddqd` — Skip to cinematic (bypass mini-game)
- `idkfa` — Skip to Venator 3D model
