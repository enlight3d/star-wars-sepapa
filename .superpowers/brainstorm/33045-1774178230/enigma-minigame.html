Énigme & Mini-jeu — Avant la révélation

Alexis doit mériter son cadeau ! Quelles épreuves lui imposer ?

L'Énigme (entre le Click to Start et l'Intro)

A

Le Code Impérial

Un terminal style Empire avec un message crypté à décoder. Le code pourrait être un truc que seul Alexis connaît (sa date de naissance, un mot de passe rigolo lié à un inside joke, ou simplement son prénom à taper).

"Identification requise. Entrez votre code d'accès, Commandant."

B

Quiz Star Wars

2-3 questions sur Star Wars que tout vrai fan connaît. Style hologramme, chaque bonne réponse déverrouille un fragment du message. Exemples :

  • "Quel est le vrai nom de Dark Vador ?"
  • "De quel type est le Venator : croiseur, destroyer, ou frégate ?"
  • "Qui a dit : 'Do or do not, there is no try' ?"
C

Puzzle de Coordonnées

Une carte galactique s'affiche. Alexis doit trouver les bonnes coordonnées hyperespace pour "sauter" vers sa surprise. Indice subtil qui pointe vers la bonne planète/système.

Le Mini-jeu (entre le crawl texte et la construction du Venator)

A

Pilote le Venator !

Un mini side-scroller / space shooter 2D style rétro. Alexis pilote un petit Venator pixel-art et doit esquiver des astéroïdes ou tirer sur des droïdes pendant 30 secondes. Style arcade, impossible à perdre (mais fun).

→ Parfait pour le côté Lego pixelisé + Star Wars

B

Attrape les briques !

Des briques Lego tombent du ciel, Alexis doit les attraper avec un panier/vaisseau en les faisant glisser. Chaque brique attrapée ajoute une pièce au Venator en arrière-plan. Objectif : attraper suffisamment de briques pour "débloquer" la construction.

→ Lien direct avec la métaphore Lego du cadeau

C

Memory des Contributeurs

Un jeu de mémoire (retourner les cartes) avec les avatars/initiales des 9 contributeurs. Retrouver toutes les paires pour révéler le message. Rapide (9 cartes = ~4-5 paires + 1 carte spéciale "Alexis").

→ Met en valeur les personnes derrière le cadeau

Tu peux sélectionner une option pour l'énigme ET une pour le mini-jeu ! Ou propose tes propres idées.
