Architecture Technique — The Full Saga

SvelteKit + Three.js (Threlte) — structure du projet

Stack Technique
Framework : SvelteKit (mode static adapter — pas de serveur)
3D : Threlte (Three.js pour Svelte)
Audio : Howler.js (gestion audio cross-browser, mobile-friendly)
Animations : Svelte transitions natives + CSS keyframes
Déploiement : Vercel / Netlify / GitHub Pages (static)
Responsive : Mobile-first, touch controls pour le mini-jeu
Architecture des Composants
src/
├── routes/
│   └── +page.svelte          ← Orchestrateur principal
│
├── lib/
│   ├── stores/
│   │   └── gameState.ts       ← État global (étape courante, score, audio)
│   │
│   ├── components/
│   │   ├── Starfield.svelte   ← Fond étoilé animé (réutilisé partout)
│   │   ├── ClickToStart.svelte
│   │   ├── ImperialTerminal.svelte  ← Énigme Code Impérial
│   │   ├── TrenchRun.svelte   ← Mini-jeu X-Wing (canvas 2D)
│   │   ├── FarAway.svelte     ← "Il y a longtemps..."
│   │   ├── LogoReveal.svelte  ← "JOYEUX ANNIVERSAIRE"
│   │   ├── CrawlText.svelte   ← Texte incliné qui défile
│   │   ├── VenatorBuild.svelte ← Construction 3D (Threlte)
│   │   └── FinalMessage.svelte ← Hologramme + noms + confettis
│   │
│   ├── game/
│   │   ├── trenchRun.ts       ← Logique du jeu (boucle, collisions)
│   │   ├── sprites.ts         ← X-Wing, obstacles (pixel art)
│   │   └── controls.ts        ← Clavier + touch mobile
│   │
│   ├── three/
│   │   ├── venator.ts         ← Modèle 3D low-poly du Venator
│   │   └── particles.ts      ← Confettis, particules dorées
│   │
│   └── audio/
│       └── audioManager.ts    ← Gestion musique + SFX
│
└── static/
    ├── fonts/                 ← Font Star Wars (Star Jedi)
    └── audio/                 ← Musique + effets sonores
